# cdots
it is a simple tool that allows to create many dotfiles such as .gitignore, travis

### Installation
```
$ go get https://github.com/onerciller/cdots
$ cd /path/to/cdots
$ make
$ go install
```

### Usage
```
Commands:

 cdots create <dotfile>
 cdots create editorconfig
 cdots create travis
 
 Options:
 
 --help, -h            show help
 --version, -v         print the version
	
```
